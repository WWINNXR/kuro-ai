# Kuro AI 🐾

Kuro AI is a LINE-based AI personal secretary that helps users manage reminders, appointments, bills, and expenses through natural conversation.

## Core Features

- Smart reminders via LINE chat
- Appointment and calendar queries
- Bill tracking
- Expense and income logging
- Daily summary support
- AI intent parsing with OpenAI
- Supabase database integration
- Deployed on Vercel

## Tech Stack

- LINE Messaging API
- Next.js
- TypeScript
- Supabase
- OpenAI API
- Vercel

## Project Goal

This project demonstrates how AI can reduce daily mental load by allowing users to manage life tasks directly inside LINE without opening multiple apps.
